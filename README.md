# Core Java Practice

This repository contains my Java practice programs. I am organizing problems based on topics so that it is easy to track what I have learned and practiced.

I started with a small set of problems earlier, but now I am restructuring everything properly and continuing my practice in a more organized way.

## Project Structure

All problems are grouped by topic:

* Arrays
* Strings
* Recursion
* Dynamic Programming
* HashMap / Data Structures
* Math
* Sorting
* Misc (mixed problems)

Each folder contains Java programs related to that specific concept.

## How I Am Using This Repository

* I solve problems regularly and place them in the appropriate folder
* Arrays problems go into the arrays package
* String problems go into the strings package
* Similarly for DP, sorting, etc.
* If a problem does not clearly belong to a category, I place it in misc

This helps me keep things clean and easy to navigate.
